Builder Controls & Widget Reference

Cheat Sheet — Laminate Me!
Drag-drop UI — no keyboard shortcuts
Top Bar (6 buttons)
✏️ BuildDrag-drop designer canvas
▶️ PlayUse your remote live over BLE
🎮 DemoTry all 12 widgets — no board needed
📦 ExportSave current design as JSON
📂 ImportLoad a JSON design file
📄 CodeGenerate MakeCode for the board
Properties Panel
📝Label — text shown on widget
🎨Model — visual style: neo, glass, neon
📏Size — width & height
🔢Min / Max — for sliders, gauges
🌈Colors — LEDs, theme accents
Input Widgets (you → micro:bit)
👆Button — press / release signal
🎚️Slider — value between Min and Max
🔘Toggle — on / off switch
🕹️Joystick — angle + distance
✛D-Pad — up / down / left / right
📍XY Pad — x + y position
⏱️Timer — start / stop / elapsed seconds
Output Widgets (micro:bit → you)
💡LED — on / off indicator
🏷️Label — text display from board
🧭Gauge — dial between Min and Max
📈Graph — live rolling line chart
🔋Battery — 0–100% level icon
BLE Connection States
Disconnected — tap "📡 Tap to Connect"
Pairing — pick "BBC micro:bit [xxxxx]"
Connected — widgets are live
Demo mode — local only, no board
Error — retry, or re-pair in OS settings
BLE UART Protocol
App → board: SET widget_id value
Board → app: UPD widget_id value
Config: GETCFG / CFGBEGIN…CFGEND
Workflow
1. Build → 2. Code (copy) → 3. Paste in makecode.microbit.org → 4. Download & flash → 5. Play → Tap to Connect. Only Chrome / Edge (desktop or Android) support Web Bluetooth.
