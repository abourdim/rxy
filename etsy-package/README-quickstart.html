Micro:bit Remote Builder

Drag-and-drop Bluetooth remote builder for the BBC micro:bit V1 or V2
v1.0.0 • Digital Download
Thank you for buying Micro:bit Remote Builder! You just grabbed a zero-install, zero-account, browser-local builder for custom Bluetooth remote controllers. Drag widgets, generate MakeCode, flash, pair, play. Everything runs on your machine — no cloud, no telemetry, no nonsense. This page walks you through the first three things to do.

First 3 Things To Do

1
Open the builder. Double-click index.html in Chrome or Edge. Tap ✏️ Build, drag widgets from the palette (Button, Slider, Joystick, Gauge…) onto the canvas. Edit Label, Size, Min/Max, Colors in the properties panel.
2
Flash the generated MakeCode. Click 📄 Code, copy the program, paste into makecode.microbit.org (JavaScript mode), hit Download, drag the .hex onto your micro:bit.
3
Pair & play. Switch to ▶️ Play, tap "📡 Tap to Connect", pick your board. Drive your widgets. No board yet? Use 🎮 Demo to try all 12 widgets offline.

Requirements

✓BBC micro:bit V1 or V2
✓Chrome 90+ or Edge 90+ on desktop or Android (Web Bluetooth)
✓Bluetooth enabled on the host phone / tablet / computer
✓USB cable or battery pack for the micro:bit
✓A free makecode.microbit.org tab (JavaScript mode) to paste & flash

Key Features

Build: drag-drop canvas • 12 widgets (7 inputs + 5 outputs) • per-widget properties (Label, Model, Size, Min/Max, Colors)

Inputs: Button • Slider • Toggle • Joystick • D-Pad • XY Pad • Timer

Outputs: LED • Label • Gauge • Graph • Battery

Connect: BLE UART text protocol • SET widget_id value / UPD widget_id value • auto-generated MakeCode • JSON export / import • neo / glass / neon themes
