Description complète

🎮 Micro:bit Remote Builder — conçois des télécommandes Bluetooth pour la BBC micro:bit sans écrire une ligne de code. Glisse des widgets sur un canevas. Tape sur Code. Colle dans makecode.microbit.org. Flashe. Appaire. Joue. Ton téléphone, ta tablette ou ton ordi devient une télécommande sans fil. CE QUE TU OBTIENS • ✏️ Canevas Build en glisser-déposer • 🎮 12 widgets — 7 entrées + 5 sorties Entrées : Bouton · Curseur · Bascule · Joystick · Croix directionnelle · Pavé XY · Minuterie Sorties : LED · Étiquette · Jauge · Graphique · Batterie • 🎨 Propriétés par widget : Label, Model, Size, Min/Max, Colors • 📄 MakeCode auto-généré — à coller et flasher • 📡 Appairage BLE UART en un tap (protocole texte SET / UPD) • 📦 Export / Import JSON de tes designs • 🎨 3 thèmes visuels : neo · glass · neon • 🎮 Onglet Demo pour tester chaque widget sans carte CONTENU DU TÉLÉCHARGEMENT • index.html — l'application builder • script.js — drag-drop + BLE + générateur MakeCode • styles.css — système visuel à trois thèmes • README.md — guide complet avec référence widgets et exemples MakeCode • etsy-package/ — imprimables (Quick-Start, Cheat Sheet, Poster, Stickers, Lesson Plan) • LICENSE — licence personnelle / mono-utilisateur classe CAS D'USAGE Commande de robots · manettes de jeu · domotique · voitures RC · instruments de musique · tout ce que tu imagines. PRÉREQUIS • BBC micro:bit V1 ou V2 + USB ou piles • Chrome ou Edge sur desktop (ou Chrome sur Android) • Bluetooth activé RESPECT DE LA VIE PRIVÉE Rien ne quitte ton ordinateur.
