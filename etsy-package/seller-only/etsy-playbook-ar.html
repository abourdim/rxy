البنية

 0–3 s    HOOK     «اِبنِ جهاز التحكم البلوتوث في 60 ث. بدون برمجة.»
 3–12 s   BUILD    سحب: Joystick + زرّان + Label على اللوحة.
12–25 s   CODE     📄 Code ← نسخ ← لصق في makecode ← فلاش.
25–40 s   PAIR     ▶️ Play ← 📡 اضغط للاتصال ← اختيار اللوحة.
40–52 s   PAYOFF   دفع الـ joystick ← الروبوت يتحرك. زر ← ضوء.
52–57 s   CTA      «الحزمة الكاملة في متجري — الرابط بالأسفل.»
57–60 s   END      الشعار والوسم متجمدان لإطار الحلقة.

تسلسل الصور (Etsy تعرض الأولى والثانية بالحجم الأكبر)

  1. البطل — الهاتف في اليد على لسان Play، يقود هدفاً مرئياً. عنوان علوي: «اِبنِ • اقْرن • العَب — بدون برمجة».
  2. لوحة Build — لقطة للبانِي مع تخطيط كامل: joystick، زرّان، gauge، label.
  3. توليد MakeCode — انقسام: البانِي يساراً، makecode.microbit.org JavaScript يميناً مع الكود الملصوق.
  4. شبكة الودجتات الـ 12 — الجميع موسومة بالإيموجي (7 مدخلات + 5 مخرجات).
  5. تدفق اقتران BLE — نافذة اختيار BBC micro:bit فوق لسان Play، سهم نحو «📡 Tap to Connect».
  6. عينة المظاهر — ثلاث لوحات: neo · glass · neon لنفس جهاز التحكم.
  7. مشهد صفي — أطفال يحملون أجهزة لوحية، واحد يقود روبوتاً صغيراً؛ ودجت Label يعرض حالة حية.
