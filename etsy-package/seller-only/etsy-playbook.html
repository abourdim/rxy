SLUG — THE 60-SECOND ARC

Structure

 0–3 s   HOOK        "Build your own Bluetooth remote in 60 seconds. No code."
 3–12 s  BUILD       Drag a Joystick + 2 Buttons + a Label onto the canvas.
12–25 s  CODE        Tap 📄 Code → copy → paste into makecode.microbit.org → flash.
25–40 s  PAIR        Switch to ▶️ Play → 📡 Tap to Connect → pick the board.
40–52 s  PAYOFF      Push the joystick → robot rolls. Press a button → light.
52–57 s  CTA         "Full kit in my shop — link below."
57–60 s  END FRAME   Logo + tagline frozen for the loop preview.

SLUG — AVOID THESE

Common mistakes

❌ Don't✅ Do
Open with "Hi, I'm…" + 10s of introOpen with the empty Build canvas and a single drag in the first 3 s
Film landscape (16:9)9:16 portrait — Etsy autoplays on mobile
Skip showing the target that movesAlways show a real thing (robot / LED / motor) react when you press a widget
Show the MakeCode screen for 20 sCut the paste/flash to <10 s — nobody wants to watch code scroll
Forget captions (Etsy plays muted)Burn in captions, highlight "No code" and "Bluetooth" hard
Show iOS / Safari in the videoAlways demo on Chrome/Edge — iPhone users will buy and return
Skip the CTASay "link below" in the last 5 seconds every single time
